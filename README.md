# Simple. Fast. Beautiful. - Koi

*24 lines of code, 3 libery, easy to use*

![koi](https://github.com/IwnuplyNotTyan/koi/blob/main/image/Showcase.png)

``` bash
koi <example.md>
```

## Setting
```bash
export THEME=dark
```
Must set to dark, light, notty or dracula
## Build from source

``` bash
go mod tidy
go build -o koi main.go
```

# Simple. Fast. Beautiful. - Koi

![koi](https://github.com/IwnuplyNotTyan/koi/blob/main/image/Showcase.png)

``` bash
koi <example.md>
```

## Setting
```bash
export THEME=dark
```
*All default theme: dark, light, notty or dracula*
# Install

## Arch, buildopkg
``` bash
makepkg -si
```

## Build from source
``` bash
go mod tidy
go build -o koi main.go
```
